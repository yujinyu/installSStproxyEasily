# installSStproxyEasily

一键安装linux全局FQ软件ss-tproxy.

使用方法：

> 1. 在conf.py中选择使用代理方式是ss还是ssr，并配置账号、密码以及其他参数；    
> 2. 运行install_ss-redir.py安装软件并进行环境配置；              
> 3. 安装配置完成后，运行命令"ss-tproxy start"开始使用。
